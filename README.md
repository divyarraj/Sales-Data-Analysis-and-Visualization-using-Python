# 📊 Sales Data Analysis and Visualization using Python

## 📌 Project Description

Analyzes retail sales data using Python to uncover trends, patterns, and customer behavior.
Visualizes insights using charts and graphs to support data-driven business decisions.

---

## 📂 Datasets Used

* `Retail_Data_Transactions.csv` → Contains transaction details such as purchases and sales
* `Retail_Data_Response.csv` → Contains customer response and behavior data

---

## 🎯 Objectives

* Analyze sales performance and trends
* Understand customer purchasing behavior
* Combine transaction and response data for better insights
* Visualize key business metrics

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib

---

## 🔍 Key Features

* Data cleaning and preprocessing
* Merging multiple datasets
* Sales trend analysis
* Customer behavior analysis
* Data visualization

---

## 📈 Visualizations

* 📉 Sales Trend Analysis (Line Chart)
* 📊 Product/Category Comparison (Bar Chart)
* 🔵 Customer Insights (Scatter Plot)

---

## ▶️ How to Run

1. Open the notebook in Jupyter Notebook
2. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib
   ```
3. Place both datasets in the project folder
4. Run all cells
